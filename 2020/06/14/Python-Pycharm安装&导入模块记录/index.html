Python-Pycharm安装&导入模块记录

加环境变量的目的是为了让系统能找到执行程序

1. Pycharm安装

  • （1）官网下载安装社区版：http://www.jetbrains.com/pycharm/download/#section=windows
  • （2）jdk版本不支持，需要升级到1.8以上；
  • （3）Pycharm使用需要激活码：

2. Python导入requests模块失败

  • 提示connect timeout，找不到模块
  • 打开cmd输入Python显示命令不存在，Python IDLE能正常使用
  • 网上搜索Python提示命令不存在的原因：没有添加环境变量。
  • 手动添加环境变量后能搜到模块，如下

    3. Python导入pip模块失败

  • pip安装时又提示timeout
  • 网上查找教程：需找到Python安装的script目录，即有pip.exe执行文件的目录，再把pip加到环境变量中，模块就能下载成功了

    注释：
    windows下要方便的使用某个命令如pip install tornado，需要把执行文件目录价到环境变量中
