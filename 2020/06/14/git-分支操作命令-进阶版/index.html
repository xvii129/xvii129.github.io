git-分支操作命令-进阶版

工作中常用的git分支操作

1. 查看文件改动：git status

2. 文件改动暂存：git stash

补充：
git stash只能缓存暂存区/git跟踪的修改，不能缓存未track的新文件/被忽略的文件

3. 撤销文件修改：git checkout / git reset

4. 分支创建/修改/删除：git checkout / git branch -m/ git branch -d

5. 其他常用操作1：删除未track文件

6. 其他常用操作2：目录下某个文件或文件夹不想提交

7. 其他常用操作3：远程分支操作
